🤖 Telegram Ranking Bot

A feature-rich Telegram bot that tracks user activity in groups, ranks users based on their engagement, and provides insightful group statistics. This bot enhances group interactions with gamification, XP leveling, leaderboards, and moderation tools.

🚀 Open RankersBot

🚀 Features

📊 User Activity Tracking

  • Tracks messages, stickers, photos, and media.
  • Records:
    • Total messages
    • XP (Experience Points)
    • Level
    • Daily streak

🏆 Ranking System

  • Leaderboards based on engagement.
  • Filters supported:
    • 📅 Today
    • 📈 This Week
    • 🏆 Forever
    • 💬 Current Chat
    • 🌐 Global Stats (coming soon)

📈 Group Statistics

  • Total messages
  • Number of users
  • Banned users
  • Active users today
  • Most active hours

📜 User History

  • Tracks:
    • Message stats over time
    • Name and username changes

🛡️ Moderation Tools

  • Detects and logs spam, links, and bad words.
  • Auto-warns, mutes, or bans based on settings.

🎮 XP & Leveling System

  • Custom XP per action:
    • Messages: +5 XP
    • Stickers: +2 XP
    • Photos: +3 XP
    • Voice: +4 XP
    • Videos: +6 XP
  • Leveling thresholds configurable.

🔘 Interactive Inline Buttons

  • 📜 Commands
  • 📊 My Stats
  • 🔄 Refresh
  • 🔝 Filter Rankings

🧾 Commands

CommandDescription
/startStarts the bot and displays intro + menu
/helpShows available commands
/rankingDisplays top 10 most active users with filters
/mytopView your XP, level, and activity
/groupstatsView detailed group statistics
/historyShows activity history of a user
/changesDisplays username and name change history

💾 Data Storage

The bot uses data.json to persist data like XP, message counts, and group stats.

{
  "groups": {
    "-1001234567890": {
      "group_name": "My Group",
      "stats": {
        "total_messages": 10000,
        "active_users_today": 20
      },
      "users": {
        "123456789": {
          "full_name": "John Doe",
          "username": "johndoe",
          "xp": 300,
          "level": 3,
          "streak_days": 2
        }
      }
    }
  }
}
